                    🚀 HOW TO GET THE MAP WORKING
================================================================================
STEP 1: Get API Key
    Visit: https://console.cloud.google.com/
    • Create new project
    • Enable "Maps SDK for Android"
    • Go to Credentials → Create API Key
    • Copy the key
STEP 2: Get Your SHA-1 Fingerprint
    Run: ./gradlew signingReport
    Copy the SHA1 value
STEP 3: Register Your App
    In Google Cloud Console:
    • Select your API Key
    • Add SHA-1 fingerprint
    • Add package: com.vasberc.mapsexample
    • Save
STEP 4: Add Key to Your App
    Edit: app/src/main/AndroidManifest.xml
    Find:
    <meta-data
        android:name="com.google.android.geo.API_KEY"
        android:value="YOUR_GOOGLE_MAPS_API_KEY_HERE" />
    Replace with your actual key:
    <meta-data
        android:name="com.google.android.geo.API_KEY"
        android:value="AIzaSy..." />
STEP 5: Build & Run
    ./gradlew clean build
    ./gradlew installDebug
    adb shell am start -n com.vasberc.mapsexample/.MapActivity
✅ DONE! Map should now display!

================================================================================
                         🆘 TROUBLESHOOTING
================================================================================
Problem: Blank/Gray Screen
    Cause: Missing API key
    Fix: Follow STEP 1-4 above
Problem: Map visible but won't respond to touches
    Cause: Permissions not granted
    Fix: Settings → Apps → Maps Example → Permissions → Grant Location
Problem: Map never loads
    Cause: No internet
    Fix: Enable WiFi or mobile data
Problem: Build fails
    Cause: Dependencies not synced
    Fix: Run ./gradlew clean build
More help: See MAP_NOT_DISPLAYING.md or TROUBLESHOOTING.md
